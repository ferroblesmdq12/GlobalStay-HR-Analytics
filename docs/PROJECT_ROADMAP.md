# Objetivo del Proyecto

Desarrollar una solución integral (End-to-End) de **HR Analytics** para una cadena hotelera multinacional ficticia, que incluya:

* Generación de datos sintéticos.
* Validación de la calidad de los datos.
* Desarrollo de procesos ETL.
* Diseño de un Data Warehouse.
* Consultas analíticas mediante SQL.
* Desarrollo de dashboards interactivos en Power BI.

El objetivo es construir un proyecto con estándares similares a los utilizados en la industria, que pueda formar parte de un portfolio profesional.
